Foundation Course

1. Basic Aari & Zardosi

• 3 Months Duration • Essential Stitches • Hand Work

Kickstart your journey with essential stitches and techniques used in Aari work.

  • Chain Stitch, Double Chain
  • Loose Running, Zig Zag, Water Filling
  • Bead Work, Chamki Work & Zardosi (and more!)
Request Syllabus

2. Advance Aari & Zardosi

• 6 Months Duration • Advanced Designs • Design Blouse Making

Upgrade your skills with advanced Aari stitches, design elements, and creative applications.

  • Wheel / Chakra, Salli Knot, Step Sequence
  • Stone Work, Mirror Work, Cut Work
  • Thread Load Emboss, Design Blouse Making (and more!)
Request Syllabus

3. Machine Embroidery

• 3 Months Duration • Basic Stitches

Build a strong foundation in machine embroidery using essential professional stitches.

  • Running Stitch, Satin Stitch, Cording Stitch
  • Fancy Stitch, Chain Stitch, Stem Stitch
  • Pin Work, Shade Work Stitch, Bead Work (and more!)
Request Syllabus

4. Machine Embroidery (Advanced)

• 6 Months Duration • Professional Level

A complete professional-level program covering beginner to advanced machine embroidery styles.

  • Shadow Work, Cut Work, Mirror Work
  • Sequence, Zari, Chinese and Quilting Stitches
  • Patch Work, Open Work, Motif Work (and more!)
